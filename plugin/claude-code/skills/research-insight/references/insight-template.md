# Research Insight Template

## 使用说明

趋势洞察报告的标准输出格式。文件名：`insight/{topic}-{YYYY-MM-DD}.md`。

**关键规则**：
1. Phase 1 问了用户关注的维度（a-g），未选的维度只用一句话带过或完全省略
2. 所有趋势判断必须有数据支撑（论文数量、被引次数）
3. "新兴方向"必须是最近 6-12 个月内首次出现的关键词/方法
4. "研究空白"必须具体到可以写 proposal 的程度
5. 方法趋势用 ↑↑/↑/→/↓/↓↓ 标记，附数据

# 研究趋势洞察 — {topic}

> 分析范围: {方向列表}
