# Paper Comparison Template

## 使用说明

多篇论文对比的标准输出格式。文件保存到 `04-对比分析/{topic}-compare-{YYYY-MM-DD}.md`。

**关键规则**：
1. 对比结论必须给出明确判断，不能只说"各有优劣"
2. 数字结果直接从 `paper_extract` 的 profile 数据填入，没有数据的格子写 `N/A`
3. 2 篇对比用优劣势表，3 篇以上用多列对比表
4. 每篇论文标题用 wikilink 指向 `02-论文库/`
