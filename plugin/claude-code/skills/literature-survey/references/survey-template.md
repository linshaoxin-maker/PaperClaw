# Literature Survey Template

## 使用说明

文献综述的标准输出格式。文件名：`survey/{topic}.md`。

**关键规则**：
1. Phase 4 问用户要包含哪些章节，未选的章节省略
2. 对比表格的维度由用户在 Phase 4 指定的"关注维度"决定
3. 每个方法分类下必须有代表性论文引用
4. 研究空白必须具体、可操作，不能泛泛而谈

## 模板

```markdown
# {topic} — 文献综述

> 覆盖时间: {start_year}—{end_year}
> 论文数量: {n} 篇
> 关键词: {keyword groups}
> 关注维度: {用户指定的分析重点}
> 生成日期: {YYYY-MM-DD}

## 1. 引言与背景

{问题定义：为什么这个研究方向重要}
{研究现状概述：目前进展到什么程度}
{本综述的覆盖范围和组织方式}

## 2. 方法分类

### 2.1 {方向/方法类别 A}

**核心思路**: {一句话概括这类方法的思路}

**代表性工作**:
- [[{paper title 1}]] ({year}) — {核心贡献}
- [[{paper title 2}]] ({year}) — {核心贡献}

**方法演进**: {从早期到最新的演进脉络}

### 2.2 {方向/方法类别 B}
...

### 2.3 {方向/方法类别 C}
...
```
