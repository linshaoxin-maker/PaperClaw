# Daily Digest Template

## 使用说明

每日阅读摘要的标准输出格式。文件名：`daily/{YYYY-MM-DD}.md`

## 模板

```markdown
# 每日阅读摘要 — {date}

## 上下文恢复

- 昨日进展: {从 workspace_context 获取的昨日活动摘要}
- 阅读进度: 待读 {n} | 阅读中 {n} | 已读 {n} | 重要 {n}
- 活跃分组: {列出有更新的分组名}

## 今日采集

| 来源 | 数量 |
|------|------|
| arXiv | {n} |
| DBLP | {n} |
| Semantic Scholar | {n} |
| 去重后总计 | {n} |
| 新增 | {n} |
| 重复跳过 | {n} |

## 高置信推荐

| # | 标题 | 评分 | 核心贡献 | 状态 |
|---|------|------|---------|------|
| 1 | [[{title}]] | {score}/10 | {一句话贡献} | {to_read/important} |
| 2 | ... | ... | ... | ... |

## 补充参考

| # | 标题 | 评分 | 简评 |
|---|------|------|------|
| 1 | ... | ... | ... |

## 今日计划

- [ ] 深入阅读: {用户选中的论文标题}
- [ ] 待跟进: {AI 建议的下一步}
- [ ] 关注方向: {与用户 profile 最匹配的今日热点}
```
